# Healthcare-Data-Dashboard

## Overview
This project presents a Tableau dashboard analyzing trends in healthcare data, including patient conditions, admissions, and billing patterns. 
The goal was to identify key trends and provide insights into patient and cost behavior.

## Dataset
The dataset includes healthcare records with:
- Patient demographics (gender, blood type)
- Medical conditions
- Admission and discharge dates
- Insurance providers
- Billing amounts

## Dashboard Features
- Patient distribution by condition and gender
- Admissions over time (trend analysis)
- Most common medical conditions
- Average billing amount by insurance provider
- Interactive filters for condition and gender

## Key Insights
- Patient admissions show noticeable seasonal spikes during warmer months
- Average billing amounts across insurance providers are relatively similar
- Certain conditions (e.g., arthritis and diabetes) show higher occurrence among female patients

## Tools
- Tableau
- Excel (data cleaning)
